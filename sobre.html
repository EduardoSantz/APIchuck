Sobre o Projeto

Este projeto consome a Chuck Norris API para exibir piadas aleatórias, com possibilidade de favoritar e gerenciar a lista.

Objetivo: Exercício prático de consumo de API pública com Fetch API.

Tecnologias: HTML, CSS, JavaScript (ES6+).

API: https://api.chucknorris.io/
